TAGRFT is a web app I built to help you find that one weirdly specific GitHub repository you've been looking for.

## How it works

Simply enter a description of what you're looking for, and TAGRFT will search through GitHub repositories to find matches.

## Features

- Natural language search
- Real-time results
- Direct links to repositories
- Free and open source

Check it out at [nellowtcs.me/TAGRFT](https://nellowtcs.me/TAGRFT)!
